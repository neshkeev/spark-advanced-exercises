# Задания из курса по Apache Spark Advanced

## Настройка

1. `docker compose build && docker compose up`
1. `docker compose exec -it pyspark bash`
1. `ssh-copy-id yarn@worker-1`. Пароль `yarn`
1. `ssh-copy-id hdfs@worker-1`. Пароль `hdfs`

## Ресурсы

Доступны следующие ресурсы:

- Namenode: http://localhost:9870
- Resource Manager: http://localhost:8087
- Jupyter Notebook: http://localhost:8888
